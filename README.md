# Тренажер ударений

React + Tailwind тренажер для быстрого повторения ударений из задания 4 ЕГЭ.

## Что умеет

- Флэшкарты: нажимаешь на гласную, на которую падает ударение.
- Отдельная игра для задания 1: тебе дают разряд, а ты сам вводишь слова из него.
- 239 слов из списка ударений уже добавлены в словарь.
- Прогресс хранится в SQLite-файле `data/progress.db`.
- Если API не запущен, приложение временно работает через `localStorage`.
- На странице `Все слова` показан процент правильности по каждому слову.
- Слова, где чаще ошибаешься, показываются выше и чаще попадают в тренировку.
- Есть режимы `Умно`, `Ошибки`, `Новые`, `Экзамен`.
- Учитывается не только правильность, но и скорость ответа, серия, давность повторения и текущая слабость слова.
- На карточке показывается причина выбора слова: новое, слабое место, пора повторить, закрепление.
- Быстрые клавиши: цифры выбирают гласную, `Enter` или `Space` переходят дальше.
- Отдельный список приоритетных слов поднимает самые важные ударения в выдаче, не добавляя новые слова в словарь.

```bash
npm install
npm run dev
```

- фронтенд: `http://127.0.0.1:5173`
- API: `http://127.0.0.1:4174`
- база прогресса: `data/progress.db`

```bash
npm run build
npm start
```

После этого сервер отдаст готовый фронтенд и API на `http://127.0.0.1:4174`.

## Как работает алгоритм выдачи

Каждое слово получает профиль памяти и вес. Чем выше вес, тем чаще слово появляется:

- новые слова получают повышенный вес;
- слова с ошибками получают сильный буст;
- слова с последней ошибкой показываются быстрее;
- слова с серией правильных ответов постепенно показываются реже;
- последние несколько карточек штрафуются, чтобы не было постоянных повторов подряд.
- быстрые правильные ответы повышают уверенность;
- медленные ответы снижают уверенность даже при правильном выборе;
- для выученных слов растет интервал повторения: минуты, часы, дни;
- если интервал прошел, слово получает статус `срочно`.

Режимы:

- `Умно` - основной режим для ежедневной тренировки.
- `Ошибки` - агрессивно поднимает слова с ошибками и низким мастерством.
- `Новые` - быстрее закрывает незнакомые слова.
- `Экзамен` - добавляет больше случайности, чтобы не привыкать к порядку.

Статистика по словам хранит:

- количество попыток;
- количество правильных ответов;
- количество ошибок;
- текущую серию;
- дату последнего ответа;
- был ли последний ответ правильным.
- среднее время ответа.

## Структура

- `src/data/stressWords.ts` - словарь ударений.
- `src/data/highYieldStressWords.ts` - приоритетные слова для повышенного ранжирования.
- `src/data/taskOneTraining.ts` - справочник для задания 1.
- `src/lib/stress.ts` - статистика, расчет точности и алгоритм выбора карточки.
- `src/lib/progressApi.ts` - клиент для API прогресса.
- `src/components/FlashcardTrainer.tsx` - экран тренировки.
- `src/components/AllWordsPage.tsx` - список всех слов.
- `src/components/TaskOneTrainer.tsx` - игра для задания 1 с вводом ответа, активным вспоминанием и возвратом ошибок.
- `server/index.mjs` - Express API и SQLite `.db` через `sql.js`.
